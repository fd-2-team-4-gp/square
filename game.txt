- Делаете гет запрос по ownerId - получаете список игр.

- Таблица log - только для post-запросов

- Для создания игры - делаете пост-запрос с указанием максимального счета

- Post-запрос в таблицу log - создает ход и возвращает результат. Туда нужно отправлять from - от кого, to - кому передается мяч

- "гет запрос по ownerId" это для таблицы game - для получения списка игр текущего игрока (edited)

- user1 назначается авторизованный пользователь, который отправляет запрос на создание игры

- для создания новой игры надо отправлять post-запрос в таблицу game.

- log - это таблица для создания действий.

- Алгоритм приблизительно такой:
    1) Отправляете post-запрос в таблицу game - возвращается созданное поле. 
    2) Если мячик у залогиненного пользователя - подсказываете, что нужно кликнуть мышкой по полю игроков. 
    3) По клику - отправляете post-запрос в таблицу log. 
    4) Если мячик у виртуального игрока - отправляете запрос только с полем "from" в поле "next" вам вернется кому был передан мяч.  
    5) Отрисовываете перемещение мяча или гол
    Ну, и дальше, до тех пор, пока игра не закончится.
